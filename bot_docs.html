Monogram Bot API

Telegram-compatible Bot API for Monogram Messenger

Authentication

All Bot API requests require authentication via API key. Your bot's API key is generated when you create a bot.

Header: X-Api-Key: YOUR_API_KEY

Query param: ?api_key=YOUR_API_KEY

Base URL: /bots/api/

Error responses follow Telegram format:

{"ok": false, "error_code": 400, "description": "Bad Request: ..."}

All Methods

Core Methods

POSTsendMessage /bots/api/sendMessage

Send a text message to a chat.

Parameters

FieldTypeDescription
chat_idIntegerRequiredUnique identifier for the target chat
textStringRequiredText of the message to be sent
reply_to_message_idIntegerOptionalIf the message is a reply, ID of the original message
curl
Python
JavaScript
curl -X POST http://localhost:8000/bots/api/sendMessage \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"chat_id": 123, "text": "Hello from bot!"}'
import requests
r = requests.post("http://localhost:8000/bots/api/sendMessage",
    headers={"X-Api-Key": "YOUR_API_KEY"},
    json={"chat_id": 123, "text": "Hello from bot!"})
print(r.json())
fetch("http://localhost:8000/bots/api/sendMessage", {
  method: "POST",
  headers: {"X-Api-Key": "YOUR_API_KEY", "Content-Type": "application/json"},
  body: JSON.stringify({chat_id: 123, text: "Hello from bot!"})
}).then(r => r.json()).then(console.log)

Response

{
  "ok": true,
  "result": {
    "message_id": 1,
    "from": {"id": 1, "is_bot": false, "first_name": "Owner"},
    "chat": {"id": 123, "type": "private"},
    "date": 1700000000,
    "text": "Hello from bot!"
  }
}

GETgetUpdates /bots/api/getUpdates

Get new updates (long polling).

FieldTypeDescription
offsetIntegerOptionalReturn only messages with ID > offset
limitIntegerOptionalMax number of updates (1-100, default 100)
timeoutIntegerOptionalLong polling timeout in seconds
curl
Python
curl "http://localhost:8000/bots/api/getUpdates?offset=0&limit=10&timeout=30" \
  -H "X-Api-Key: YOUR_API_KEY"
import requests, time
offset = 0
while True:
    r = requests.get("http://localhost:8000/bots/api/getUpdates",
        params={"offset": offset, "timeout": 30},
        headers={"X-Api-Key": "YOUR_API_KEY"})
    data = r.json()
    if data["ok"]:
        for update in data["result"]:
            print(update)
            offset = update["update_id"] + 1
    time.sleep(1)

Update Format

Updates from getUpdates or webhook follow this structure:

{
  "update_id": 123,
  "message": {
    "message_id": 1,
    "from": {"id": 1, "is_bot": false, "username": "user", "first_name": "User"},
    "chat": {"id": 1, "type": "private", "title": "Chat Name"},
    "date": 1234567890,
    "text": "Hello"
  }
}

Error Format

All errors return a JSON response with ok: false:

{"ok": false, "error_code": 400, "description": "Bad Request: chat_id and text are required"}
CodeDescription
400Bad Request - missing or invalid parameters
401Unauthorized - missing or invalid API key
404Not Found - resource not found
429Too Many Requests - rate limited
